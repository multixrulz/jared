Fixing the ROX Filer mount-point icon

ROX Filer is an awesome file manager for Linux (and other unixes). The only complaint I’ve had recently is its icon for mount-points. This is because I’ve been playing around with some pretty icon themes, but those themes never included the file name that ROX Filer expects to find for its mount-points. ROX does allow you to set icons per directory or mime-type – but then you can’t change the theme.

The solution is simple. Copy the directory icon to /path/to/your/theme/scalable/mimetypes/inode-mount-point.png.

Woohoo! Now mount-points in ROX look as cool as the rest of your icon theme.
