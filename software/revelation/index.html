Revelation

Revelation is presentation software. There’s a lot of presentation software out there, so why another one?

  • I need to be able to create slides real quick!. This means the best format is plain text.
  • I need to be able to style slides, and be able to quickly change that style globally.
  • I like my data to be secure. (I have had software eat my data, not fun).
  • It must be Free Software.
  • I won’t touch anything written in Java.

Don’t get me wrong, I like some of the alternatives, and use them where appropriate. But when I need to put up words for songs, and similar uses, they just don’t cut it for me.

Revelation is still a baby, and not in a state where I want to distribute it yet.
